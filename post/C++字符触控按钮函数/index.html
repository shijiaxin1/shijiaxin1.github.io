一、设置按钮

  1. 一个按钮 void ButtonSet(int Button_ID,__STYLE Button_type,__STYLE Button_line,int Button_X,int Button_Y,int Button_Width,int Button_Height);

  2. 一行按钮 void ButtonLine(int start_X,int start_Y,int button_NUM,int button_W,int button_H,int button_SPACE,__STYLE line_STYLE,__STYLE button_DIRECTION);

  3. 一组按钮矩阵 void ButtonRect(int startX,int startY,int numW,int numH,int buttonW,int buttonH,int spaceW,int spaceH,STR_TABLE buttonTEXT/*Form [0]*/,__STYLE lineSTYLE);

二、显示按钮

  1. 一个按钮 void ButtonShow(int Button_ID);
  2. 所有按钮 void ButtonShowAll(void);

三、等待按钮被点击

int ButtonTouch(void);

【注意事项】：

  1. 如果你想自己绘制按钮，但想一同接受点击，也可以使用本函数，可以先使用 ButtonSet() ，但不进行 ButtonShow()；
  2. 如果你想绘制一个按钮矩阵，但想隐去其中的几个按钮，也可以使用本函数，只需将想隐去的按钮的 __BUTTON_used 赋值为 0
